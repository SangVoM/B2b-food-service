# B2b food service

This is the test application for the SmartDev company

##Technologies using in this project

- Front-end side
    - Reactjs
    - Ant design
    - Redux toolkit
    - Node sass
- Back-end side
    - Nodejs
    - Express
    - Mongodb
    - Morgan (Handle request logging)
    - Yup for the validation schema
- Deployment
    - Heroku
- Code formatting
    - Prettier
- IDE
    - Webstorm

## How to run this project

For running the front-end application

`cd client/`

`npm install`

`npm run dev`

For running the Back-end server

`cd server/`

`npm install`

`npm run dev`
